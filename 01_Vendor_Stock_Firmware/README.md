### 📂 Folder Contents (Structure):

2. **USB Upgrade:** The easiest way to update the system via a flash drive while the device is operational.

⚠️ **Attention / Внимание:** Ensure your device is exactly **Vontar X3** before flashing. Using these files on other devices may cause a brick.
*Перед прошивкой убедитесь, что ваше устройство — именно **Vontar X3**. Использование этих файлов на других приставках может привести к «окирпичиванию».*
