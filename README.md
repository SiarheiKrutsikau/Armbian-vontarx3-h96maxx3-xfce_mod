# Armbian Vontar X3 / H96 Max X3 (XFCE Mod)
Custom Armbian 24.8.2 Bullseye (Kernel 5.15) optimized for Vontar X3 and H96 Max X3 TV Boxes. This project aims to provide a stable desktop experience with full hardware support.

## 🚀 Key Features
* 🖥️ **XFCE Desktop:** Lightweight and pre-configured environment.
* 📟 **LCD Support:** Working front panel display support for Vontar X3.
* 🛠️ **Auto-eMMC Script:** Automated formatting and installation to internal memory.
* 🔧 **Optimized DTB:** Specifically tuned using H96Max X3 device tree.

## 📂 Repository Structure
* **[01_Vendor_Stock_Firmware](./01_Vendor_Stock_Firmware):** Original factory images (PC & USB) to restore your device.
* **[02_Base_Armbian](./02_Base_Armbian):** Stable clean base image (Kernel 5.15) used for this build.
* **[03_My_Custom_Project](./03_My_Custom_Project):** My personal scripts, UI tweaks, and modification code.
* **[04_Documentation](./04_Documentation):** Technical notes and hardware information.

## 📥 Downloads
All ready-to-use images are available in the **[Releases](https://github.com/SiarheiKrutsikau/Armbian-vontarx3-h96maxx3-xfce_mod/releases)** section.

# Armbian Vontar X3 / H96 Max X3 (XFCE Mod) — на русском
Кастомная сборка Armbian 24.8.2 Bullseye (Ядро 5.15), оптимизированная для ТВ-боксов Vontar X3 и H96 Max X3. Цель проекта — стабильная работа графической оболочки с полной поддержкой железа.

## 🚀 Особенности
* 🖥️ **Рабочий стол XFCE:** Легкая и предварительно настроенная графическая среда.
* 📟 **Поддержка LCD:** Исправлена работа дисплея передней панели для Vontar X3.
* 🛠️ **Скрипт Auto-eMMC:** Автоматическое форматирование и установка системы во внутреннюю память.
* 🔧 **Оптимизированный DTB:** Специально настроенное дерево устройств на базе H96Max X3.

## 📂 Структура проекта
* **[01_Vendor_Stock_Firmware](./01_Vendor_Stock_Firmware):** Оригинальные заводские образы (для ПК и USB) для восстановления устройства.
* **[02_Base_Armbian](./02_Base_Armbian):** Чистый базовый образ (Ядро 5.15), на котором строится этот мод.
* **[03_My_Custom_Project](./03_My_Custom_Project):** Мои личные скрипты, настройки интерфейса и код модификации.
* **[04_Documentation](./04_Documentation):** Технические заметки и информация по железу.
